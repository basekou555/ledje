# 00_inbox — captures brutes à trier

**Rôle** : la boîte d'entrée du cerveau. Tout ce qui arrive du terrain (entretien, dégustation,
visite, appel, idée) est capturé ici **brut et daté**, sans obligation de le ranger à chaud.
Le tri vient plus tard — c'est ce qui rend la capture systématique possible.

## Règles du dossier

- Une capture = un événement. Gabarit : [`../05_systeme/templates/template-debrief-terrain.md`](../05_systeme/templates/template-debrief-terrain.md).
- Nommage : `AAAA-MM-JJ-sujet.md` — **seul dossier où la date est dans le nom de fichier**.
- Faits bruts et verbatims séparés des interprétations (discipline Mom Test).
- Tri sur demande (« trie l'inbox ») ou quand une session touche un domaine ayant des captures
  en attente : chaque élément est dispatché vers sa fiche, puis la capture est supprimée.
  Une capture non triable reste, avec la raison notée dedans.
- Un inbox vide est l'état normal. Un inbox qui gonfle est un signal d'alerte.

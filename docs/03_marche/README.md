# 03_marche — face au client

**Rôle** : tout ce qui touche le marché et le client — les canaux de vente et d'acquisition,
le site comme machine à convertir, le contenu et les pubs, la recherche utilisateur. C'est
le dossier qui bouge le plus vite avec le terrain.

## Règles du dossier

- **Zone LIBRE** : l'IA met à jour statuts et suivis au fil du travail, en le signalant.
- Tout texte orienté client produit depuis ces fiches passe la grille de `../01_adn/conformite.md`.
- Nouveau canal : gabarit [`../05_systeme/templates/template-fiche-canal.md`](../05_systeme/templates/template-fiche-canal.md)
  (⚠️ plancher de prix = interne, jamais diffusé).
- Le pipeline du quotidien (relances, statuts d'appels) vit dans Notion ; ici, l'état durable.

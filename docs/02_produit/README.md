# 02_produit — le produit physique et son cadre

**Rôle** : ce qu'on vend et ce qui l'encadre — le cristal de miel, la bouteille, le prix,
l'étiquetage réglementaire. La vérité produit vit ici ; le discours qui la vend vit dans
`../03_marche/`.

## Règles du dossier

- **Zone INTERMÉDIAIRE** : modification proposée par l'IA dans sa réponse, appliquée après
  OK de Basekou (une spec produit engage le sourcing et la conformité).
- Jamais de spec inventée : un point non confirmé fournisseur reste balisé « à compléter ».
- Terminologie verrouillée : « cristal de miel » / « eau miellée » (cf. `../01_adn/conformite.md`).
- Nouvelle fiche produit : gabarit [`../05_systeme/templates/template-fiche-produit.md`](../05_systeme/templates/template-fiche-produit.md).

# 01_adn — ce qui définit Lédjé

**Rôle** : l'identité du projet — vision, positionnement, combats, cibles, marque, conformité,
identité visuelle. C'est la zone qui bouge le moins et qui contraint tout le reste : avant
d'écrire un texte, de compiler un prompt ou d'ouvrir un canal, c'est ici qu'on vérifie.

## Règles du dossier

- **Zone PROTÉGÉE** : modification uniquement sur demande explicite de Basekou, jamais
  suggérée spontanément (cf. `../05_systeme/workflow-maj.md`).
- Toute production de texte sortant passe la grille de `conformite.md` — étape fixe.

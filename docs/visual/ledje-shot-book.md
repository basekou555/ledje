---

## Blocs réutilisables (DA — Direction Artistique)

Au lieu de répéter la même info dans chaque fiche, on référence un bloc :

- **DA-01** = lumière naturelle douce (matin diffus ou fin d'après-midi), jamais de flash/studio visible.
- **DA-02** = visage jamais identifiable, yeux jamais visibles ; hors champ si possible.
- **DA-03** = sobriété vivante — pas de minimalisme froid, pas de vocabulaire luxe ; une trace de mouvement/imperfection dans le cadre.
- **DA-04** = règle des 3 éléments maximum, espace négatif généreux (pour intégrer du texte plus tard).

---

## Structure d'une fiche

`ID` · `Nom` · `⭐ Iconique ?` · `Objectif` · `Sujet/Action (le delta narratif, propre à ce shot)` · `Recette (IDs choisis dans ledje-prompt-library.md)` · `Notes/contraintes spécifiques`

**Recette** = les options choisies dans `ledje-prompt-library.md`, sous la forme `DEC-.. / CAM-.. / LIGHT-.. / MAT-.. / COMP-.. / ACT-.. / EMO-..`. C'est ce qui permet d'identifier vite une combinaison et de la retrouver dans `ledje-asset-log.md`. Sauf mention contraire, tous les shots ci-dessous appliquent **DA-01, DA-03, DA-04** (cf. `ledje-visual-language.md`) ; **DA-02** s'applique dès qu'une présence humaine (main, silhouette) est visible.

## Méthode de production (rappel, cohérent avec SOT §6.1)

Pour chaque shot : partir de la Recette (IDs de `ledje-prompt-library.md`, ou le preset de la catégorie du shot), l'ajuster si Basekou le demande, puis compiler le prompt bloc par bloc (`SCENE → PRODUCT → COMPOSITION → CAMERA → LIGHT → MATERIALS → ÉMOTION`, structure définie dans `ledje-visual-language.md`). Valider sur le papier avant de dépenser un crédit Higgsfield. Un changement de variable à la fois. Commencer par **H01** (l'image la plus importante, socle des autres).
